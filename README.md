# Contact-Management-System
Contact Management System created using python 
A Python-based Object-Oriented application for managing phone contacts with add, search, display, and validation features. Perfect for demonstrating OOP principles, data handling, and input validation skills.

Features
Add Contacts: Input name and phone number with real-time validation

Display All: View complete contact list with formatted output

Search: Case-insensitive contact lookup by name

Phone Validation: Validates 10-digit Indian phone numbers with cleaning

OOP Design: Class methods, static methods, and dictionary-based storage
